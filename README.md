Todo List App

Description

This is a simple Todo List application built with React using the Vite build tool.
The app displays a list of todos on the screen using React components and the JavaScript map() function.

Installation

Clone the repository:

git clone https://github.com/Seanwarren77/todo-list.git

Navigate into the project folder:

cd todo-list

Install dependencies:

npm install
Run the Development Server

Start the Vite development server:

npm run dev

Open the application in your browser at:

http://localhost:5173
